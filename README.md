# vue-elemnet-template

## 构建步骤
```
cnpm install -g @vue/cli                全局安装cli3脚手架
vue create vue-element-template         创建文件夹
cd vue-element-template                 进入文件夹
vue add element                         引入element-ui
```

### 下载源码运行步骤
```
npm install         
npm run serve       
```

### 打包
```
npm run build       
```
